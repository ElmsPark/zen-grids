A responsive grid system built with the power of Sass

Reference documentation

This documentation is for the pre-release, next version of Zen Grids. The API for Zen Grids 2.x is currently unstable since it is still an alpha release.

For the stable Zen Grids 1.x documentation, see zengrids.com.

Examples #

Examples of types of layouts that Zen Grids can accomplish.

Example 1 #

A
 
B
C
$zen-columns: 7; // Set the total number of columns in the grid.
$zen-gutters: 10px;

.example-container {
  @include zen-grid-container(); // Apply this mixin to the container.
}
.block-a {
  @include zen-grid-item(4, 1);
}
.block-b {
  @include zen-grid-item(3, 1, right);
}
.block-c {
  @include zen-new-row(right);
  @include zen-grid-item(3, 1, right);
}
